pagarme-php
===========

Pagar.me's PHP API

[![Build Status](https://travis-ci.org/pagarme/pagarme-php.png?branch=master)](https://travis-ci.org/pagarme/pagarme-php)

Requisitos
==========

- PHP > 5.3
- Ext. Curl
- Ext. json
- Ext. bmstring


## Instalação

```

composer.phar install

```

## Testando

```
php php tests/Pagarme.php
```
